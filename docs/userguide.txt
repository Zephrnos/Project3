Zip Code Project 3 User Guide

This guide will explain how to setup the directory, and use the command line to 
compile and run our project.  It will also go over how to use all of the 
supported commands.

--------------------------
1. Setting Up Directory
--------------------------
Your folder should look like this:

    Project3/
    ├── src/
    │   ├── main.cpp
    │   ├── BSSBlock.cpp
    │   ├── BSSFile.cpp
    │   ├── BSSIndex.cpp
    │   ├── convertCSV.cpp
    │   ├── IndexManager.cpp
    │   └── readBinaryFile.cpp
    ├── headers/
    │   ├── BSSBlock.h
    │   ├── BSSFile.h
    │   ├── BSSFileHeader.h
    │   ├── BSSIndex.h
    │   ├── convertCSV.h
    │   ├── HeaderBuffer.h
    │   ├── IndexManager.h
    │   └── ZipCodeRecordBuffer.h
    ├── data/
    │   ├── us_postal_codes.csv
    │   ├── converted_postal_codes.csv
    │   ├── new_postal_codes.csv
    │   └── newBinaryPCodes.dat
    ├── zipCodes.bss
    └── zipCodes.bss.idx

If your layout does not match this, create the folders manually:
    
    mkdir src headers data

and then copy the submitted header, data, and source files into
the proper folder.

------------------------------------
2. Building and Running the program
------------------------------------

First, make sure your current directory is at the project root.

To build on unix machines:

    g++ -std=c++17 src/*.cpp -I headers -o Project3

To build on windows machines:

    g++ -std=c++17 src\*.cpp -I headers -o Project3.exe

To run the program:

    ./Project3        (unix)
    Project3.exe      (windows)

When running without arguments, the program will perform a demo that includes 
creating the BSS file if missing, performing predefined searches, dumping the 
BSS file, finding extreme zip codes, and starting interactive mode.

Usage examples:

    ./Project3 -i
        Start interactive zip code lookup mode

    ./Project3 --test
        Run search test demonstration with valid and invalid zip codes

    ./Project3 --test-add
        Run record addition test (Phase 1: block splitting)

    ./Project3 <bss_file> -Z<zip1> [-Z<zip2> ...]
        Search for specific zip codes

    ./Project3
        Run in demo mode with predefined searches and then interactive mode

Arguments:

    -i, --interactive  Start interactive mode
    --test             Run search test demonstration
    --test-add         Run record addition test (Phase 1)
    <bss_file>         Path to the blocked sequence set file
    -Z<zipcode>        Zip code to search for (e.g., -Z10001)

Examples:

    ./Project3 -i
    ./Project3 --test
    ./Project3 --test-add
    ./Project3 Data/zipCodes.bss -Z10001
    ./Project3 Data/zipCodes.bss -Z10001 -Z90210 -Z60601

------------------------------------
3. Resetting Output Files
------------------------------------

To start fresh:

    rm -f zipCodes.bss zipCodes.bss.idx

Then regenerate by running the program with demo, interactive, or test mode:

    ./Project3
    ./Project3 -i
    ./Project3 --test
    ./Project3 --test-add
